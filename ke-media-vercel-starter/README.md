
# KẾ Media – One‑Page Portfolio (Next.js)

## Run locally
```bash
npm install
npm run dev
```
Visit http://localhost:3000

## Deploy to Vercel
1. Push folder to a GitHub repo (or upload).
2. Go to https://vercel.com/new → Import the repo.
3. Framework: **Next.js**. No extra env needed.
4. Deploy → you get a public URL.
